Welcome to the official repository for **Robo Rumble '26** – the ultimate techfest battleground! This is a modern, high-performance web application built to showcase event details, manage team registrations, and deliver an immersive, futuristic visual experience.

---

## 🚀 Key Features

* **Futuristic UI/UX:** A sleek, modern, dark-themed dashboard with vibrant neon accents.
* **3D Interactive Elements:** Immersive visual experience and interactive animations powered by Three.js / React Three Fiber.
* **Dynamic Event Cards:** Interactive sections displaying event rules, schedules, and battle details.
* **Smooth Registration:** Seamless and responsive registration forms for participants and tech teams.
* **Fully Responsive:** Optimized flawlessly for mobile, tablet, and desktop screens.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router)
* **Styling:** Tailwind CSS (with custom neon glow configurations)
* **3D Graphics:** Three.js / React Three Fiber
* **Fonts:** Optimized via `next/font` (Geist Font Family)
